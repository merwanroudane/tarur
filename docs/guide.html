Category 1

ESTAR-Based Unit Root Tests

All four tests derive from a Taylor expansion of the ESTAR model: $y_t = y_{t-1} + \phi y_{t-1}\,[1 - e^{-\theta y_{t-1}^2}] + \varepsilon_t$, where under $H_0: \theta=0$ the process is a unit root and under $H_1: \theta>0$ it is globally stationary with nonlinear mean reversion.

1 KSS (2003) — Nonlinear ESTAR Unit Root Test

Kapetanios, G., Shin, Y., & Snell, A. (2003). Testing for a unit root in the nonlinear STAR framework. Journal of Econometrics, 112(2), 359–379.

Taking a first-order Taylor expansion of the transition function around $\theta=0$ gives the auxiliary regression:

$$\Delta y_t = \beta_1 y_{t-1}^3 + \sum_{i=1}^{p} \rho_i \Delta y_{t-i} + u_t$$

Test statistic: $t_{NL} = \hat{\beta}_1 / \text{se}(\hat{\beta}_1)$ — a standard $t$-ratio on the cubic term. Reject $H_0$ when $t_{NL} < \text{CV}$ (left-tailed).

Case specification: raw — no adjustment; demeaned — subtract OLS mean; detrended — subtract OLS linear trend. The demeaned case is most common for exchange rates and commodity prices.

2 Kruse (2011) — Modified Wald τ Test

Kruse, R. (2011). A new unit root test against ESTAR based on a class of modified statistics. Statistical Papers, 52(1), 71–85.
⚠ Bug Fix: R NonlinearTSA uses standard Chi-squared — TARUR implements the correct modified Wald τ

KSS restricts the ESTAR location parameter $c=0$. Kruse relaxes this, giving auxiliary regression:

$$\Delta y_t = \beta_1 y_{t-1}^3 + \beta_2 y_{t-1}^2 + \sum_{i=1}^{p} \rho_i \Delta y_{t-i} + u_t$$

The test statistic is not a standard Wald. It is the modified Wald:

$$\tau = t^2_{\hat{\beta}_{2\perp}=0} + \mathbf{1}(\hat{\beta}_1 < 0)\cdot t^2_{\hat{\beta}_1=0}$$

The indicator $\mathbf{1}(\hat{\beta}_1 < 0)$ implements the one-sided alternative. Reject $H_0$ when $\tau > \text{CV}$ (right-tailed).

3 Sollis (2009) — Asymmetric ESTAR Test

Sollis, R. (2009). A simple unit root test against asymmetric STAR nonlinearity. Economic Modelling, 26(1), 118–125.
⚠ Bug Fix: R code uses wrong auxiliary regression — TARUR implements the correct AESTAR formulation

Extends ESTAR to allow asymmetric adjustment: upward and downward departures from equilibrium revert at different speeds. Auxiliary regression:

$$\Delta y_t = \phi_1 y_{t-1}^3 + \phi_2 y_{t-1}^4 + \sum_{i=1}^{p} \kappa_i \Delta y_{t-i} + \eta_t$$

Two statistics are computed:

  • $F_{AE}$: Joint $F$-test of $H_0: \phi_1=\phi_2=0$ — the unit root test (reject when $F_{AE} > \text{CV}$)
  • $F_{as}$: $F$-test of $H_0: \phi_2=0$ — symmetry test (reject $\Rightarrow$ asymmetric mean reversion)
result = tarur.sollis2009_test(y, case='demeaned', max_lags=12)
print(f"Symmetry Fas: {result.extra['Fas']:.3f}  p={result.extra['Fas_pvalue']:.4f}")
print(result.extra['symmetry_test'])

4 Hu & Chen (2016) — 3-Parameter Modified Wald

Hu, J., & Chen, Z. (2016). A unit root test against globally stationary ESTAR models when local condition is non-stationary. Economics Letters, 146, 89–94.
⚠ Bug Fix: R code applies same error as Kruse — TARUR uses correct 3-parameter orthogonalized Wald

Allows a locally non-stationary (explosive) but globally stationary ESTAR process. Auxiliary regression adds the linear term:

$$\Delta y_t = \beta_1 y_{t-1} + \beta_2 y_{t-1}^2 + \beta_3 y_{t-1}^3 + \sum_{i=1}^{p} \rho_i \Delta y_{t-i} + u_t$$

The 3-parameter modified Wald statistic (right-tailed) has higher power than KSS when the data-generating process is locally explosive.

result = tarur.hu_chen_test(y, case='demeaned', max_lags=12)
Category 2

Smooth Transition Unit Root Tests

These tests first estimate a nonlinear smooth transition deterministic trend via NLS, subtract it, then apply an ADF test to the residuals. Each test has its own distinct critical values from its original paper — they are not interchangeable.

5 LNV (1998) — Single Logistic Smooth Transition

Leybourne, S., Newbold, P., & Vougas, D. (1998). Unit roots and smooth transitions. JTSA, 19(1), 83–97.

The logistic transition function: $S_t(\gamma,\tau) = \bigl[1 + \exp(-\gamma(t - \tau T))\bigr]^{-1}$ smoothly shifts from 0 to 1. Three models:

$$\text{Model A: }\; y_t = \alpha_1 + \alpha_2 S_t + \nu_t$$
$$\text{Model B: }\; y_t = \alpha_1 + \beta_1 t + \alpha_2 S_t + \nu_t$$
$$\text{Model C: }\; y_t = \alpha_1 + \beta_1 t + \alpha_2 S_t + \beta_2 t\, S_t + \nu_t$$

NLS estimates $\gamma$ and $\tau$, then the ADF $t$-statistic on the detrended residuals $\hat{\nu}_t$ is compared to LNV Table I (not ADF tables!).

6 Vougas (2006) — Extended Single Transition (5 Models)

Vougas, D. (2006). On unit root testing with smooth transitions. Computational Statistics & Data Analysis, 51(2), 797–800.

Extends LNV with five functional forms (Models A–E), including trend-only transition and exponential transition. Each model has a distinct CV table. Model A mirrors LNV but with updated CVs from Vougas (2006) Table 1.

for m in ['A','B','C','D','E']:
    r = tarur.vougas_test(y, model=m, max_lags=12)
    print(f"Model {m}: t={r.statistic:.3f}, CV(5%)={r.critical_values.cv5:.3f}, {r.decision}")

7 Harvey & Mills (2002) — Double Smooth Transition

Harvey, D.I., & Mills, T.C. (2002). Unit roots and double smooth transitions. J. Applied Statistics, 29(5), 675–683.

Allows two smooth transitions (two structural breaks). The double transition function is:

$$y_t = \alpha_1 + \alpha_2 S_t^{(1)} + \alpha_3 S_t^{(2)} + \nu_t \ \ \text{(Model A)}$$

The CVs are substantially larger in absolute value than LNV because the more flexible double-transition alternative shifts the null distribution. Using LNV CVs here would lead to over-rejection.

Category 3

Asymmetric / TAR Unit Root Tests

Threshold autoregressive models allow the speed of adjustment to depend on whether the deviation from equilibrium is positive or negative — a key feature of many commodity prices and real exchange rates.

8 Enders & Granger (1998) — MTAR Test

Enders, W., & Granger, C.W.J. (1998). Unit-root tests and asymmetric adjustment with an example using the term structure of interest rates. JBES, 16(3), 304–311.

The Momentum Threshold Autoregression (MTAR) model uses the sign of $\Delta y_{t-1}$ as the threshold indicator:

$$I_t = \begin{cases} 1 & \text{if } \Delta y_{t-1} \geq 0 \\ 0 & \text{if } \Delta y_{t-1} < 0 \end{cases}$$
$$\Delta \hat{u}_t = I_t \rho^+ \hat{u}_{t-1} + (1-I_t) \rho^- \hat{u}_{t-1} + \sum_{i=1}^{p} \delta_i \Delta \hat{u}_{t-i} + \varepsilon_t$$

$\Phi$ statistic: Joint $F$-test $H_0: \rho^+=\rho^-=0$ (right-tailed). A second symmetry test checks $H_0: \rho^+=\rho^-$.

result = tarur.enders_granger_test(y, case='demeaned', max_lags=12)
print(f"rho+ = {result.extra['rho_pos']:.4f}, rho- = {result.extra['rho_neg']:.4f}")
print(f"Symmetry F = {result.extra['F_symmetry']:.3f}, p = {result.extra['F_sym_pvalue']:.4f}")

9 Sollis (2004) — ST-TAR Test

Sollis, R. (2004). Asymmetric adjustment and smooth transitions. Journal of Time Series Analysis, 25(3), 409–417.

Combines logistic smooth transition detrending with TAR-type asymmetric adjustment. The smooth transition removes a structural break in the deterministic component before testing for a unit root.

result = tarur.sollis2004_test(y, model='A', max_lags=12)

10 Cook & Vougas (2009) — ST-MTAR Test

Cook, S., & Vougas, D. (2009). Unit root testing against an ST-MTAR alternative. Applied Economics, 41(11), 1397–1404.

Combines smooth transition detrending with MTAR asymmetric adjustment. The critical values are from Cook & Vougas (2009) Table 1 — they are substantially different from Sollis (2004) and must not be shared.

Category 4

Grid-Search Tests

Rather than fixing the transition parameter, these tests search over a grid of values and take the infimum of the $t$-statistic, which gives higher power against ESTAR alternatives.

11 Kılıç (2011) — inf-t Grid Search

Kılıç, R. (2011). Testing for a unit root in a stationary ESTAR process. Econometric Reviews, 30(3), 274–302.

For each $\gamma$ in a grid $\Gamma$, the KSS $t$-statistic is computed. The test statistic is the infimum:

$$\inf_{\gamma \in \Gamma}\; t_{NL}(\gamma)$$

Because of the grid search, the null distribution differs from KSS — the critical values are larger in absolute value.

result = tarur.kilic_test(y, case='demeaned', max_lags=12)

12 Park & Shintani (2016) — Transitional AR

Park, J.Y., & Shintani, M. (2016). Testing for a unit root against transitional autoregressive models. International Economic Review, 57(2), 635–664.

A more general grid-search approach that nests the ESTAR model and allows a broader class of transition functions. Asymptotic critical values are tabulated in Park & Shintani (2016) Table 1.

result = tarur.park_shintani_test(y, case='raw', max_lags=12)
Category 5

Extended Tests

13 Pascalau (2007) — Asymmetric NLSTAR

Pascalau, R. (2007). Unit root tests with smooth breaks: an application to the Nelson-Plosser data set. Working Paper.

Extends the ESTAR auxiliary regression with additional polynomial terms ($y^4$), allowing for richer asymmetric dynamics:

$$\Delta y_t = \beta_1 y_{t-1}^2 + \beta_2 y_{t-1}^3 + \beta_3 y_{t-1}^4 + \sum \rho_i \Delta y_{t-i} + u_t$$

Joint $F$-test $H_0: \beta_1=\beta_2=\beta_3=0$ (right-tailed). Simulated CVs: 1%: 9.35, 5%: 6.82, 10%: 5.68.

result = tarur.pascalau_test(y, case='demeaned', max_lags=12)

14 Cuestas & Garratt (2011) — Cubic Polynomial Detrending

Cuestas, J.C., & Garratt, D. (2011). Is real GDP per capita a stationary process? Smooth transitions, nonlinear trends and unit root testing. Applied Economics, 43(11), 1431–1437.

Fits a cubic polynomial trend $\mu(t) = c_0 + c_1 t + c_2 t^2 + c_3 t^3$, then applies a KSS-type test to the detrended series. The $\chi^2$ statistic tests $H_0: \beta_1=\beta_2=0$ in the detrended auxiliary regression.

result = tarur.cuestas_garratt_test(y, max_lags=12)

15 Cuestas & Ordóñez (2014) — NLS Logistic Detrending + KSS

Cuestas, J.C., & Ordóñez, J. (2014). Smooth transitions, asymmetric adjustment and unit roots. Applied Economics Letters, 21(14), 969–972.

First removes a smooth transition trend via NLS (logistic form), then applies the standard KSS test to the residuals. The paper shows that KSS critical values remain valid for the NLS-detrended residuals.

result = tarur.cuestas_ordonez_test(y, max_lags=12)
Category 6

Nonlinear Cointegration Tests

16 KSS Nonlinear Cointegration (2006)

Kapetanios, G., Shin, Y., & Snell, A. (2006). Testing for cointegration in nonlinear smooth transition error correction models. Econometric Theory, 22(2), 279–303.

Estimate the static cointegrating regression $y_{1t} = \alpha + \beta y_{2t} + u_t$, then apply the KSS test to $\hat{u}_t$. Under nonlinear cointegration, the equilibrium error is a globally stationary ESTAR process rather than a linear AR(1).

result = tarur.kss_cointegration_test(y1, y2, case='demeaned', max_lags=12)

17 Enders & Siklos (2001) — TAR Cointegration

Enders, W., & Siklos, P.L. (2001). Cointegration and threshold adjustment. JBES, 19(2), 166–176.

Tests cointegration where the equilibrium error follows a TAR or MTAR process — adjustment toward equilibrium is faster in one direction than the other. The $\Phi^*$ statistic tests $\rho^+=\rho^-=0$.

result = tarur.enders_siklos_test(y1, y2, max_lags=12)
Category 7

Linearity & Diagnostic Tests

18 Teräsvirta (1994) — Linearity Test

Teräsvirta, T. (1994). Specification, estimation, and evaluation of smooth transition autoregressive models. JASA, 89(425), 208–218.

Tests $H_0$: linearity against a STAR alternative via a third-order Taylor expansion. The sequential procedure also distinguishes between LSTAR and ESTAR:

$$H_{01}: \delta_3=0,\quad H_{02}: \delta_2=0|\delta_3=0,\quad H_{03}: \delta_1=0|\delta_2=\delta_3=0$$
  • If $H_{02}$ is rejected most strongly → ESTAR
  • If $H_{01}$ or $H_{03}$ rejected most strongly → LSTAR
result = tarur.terasvirta_test(np.diff(y), d=1)
print(f"p-value: {result.extra['p_linearity']:.4f}")
print(f"Suggested: {result.extra['suggested_model']}")
print(f"H01 F={result.extra['H01']:.3f}, H02 F={result.extra['H02']:.3f}, H03 F={result.extra['H03']:.3f}")
